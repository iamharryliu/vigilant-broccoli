# Terraform

Provisions the project's cloud resources — Cloudflare DNS/Pages, Oracle Cloud VMs, Supabase auth, and GitHub repo config — plus the Packer-built VM image.

## Stack

- Language
  - HCL
  - Bash
- Tooling
  - Terraform
  - Packer
- Cloud providers
  - Cloudflare
  - Google Cloud
  - Oracle Cloud
  - Supabase
  - GitHub
- Secrets
  - HashiCorp Vault
