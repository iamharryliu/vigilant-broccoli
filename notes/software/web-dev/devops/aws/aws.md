# AWS

## CLI

CLI requires AWS user access key, secrey access key, region.

### AWS CLI

Setup and Configuration

```
brew install awscli
aws configure
```

Commands

```
aws iam list-users
```

## Services

- EC2 - cloud server, pay for the time running the server
- S3 - storage service
- Lambda - serverless functions, pay for computation time
