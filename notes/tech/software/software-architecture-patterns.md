- `package.json` - Starting point of application. Should be able to manage everything in the infrastructure.
- CI Actions
  - cron-commands
  - deploy
  - test-security
  - test-integration
  - manual versions of commands
  - notification-commands
- docs
  - STACK.md
  - INFRASTRUCTURE.md
  - DEVELOPER.md
- integrations
