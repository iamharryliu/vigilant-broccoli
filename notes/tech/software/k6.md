# K6 Load Tests

```
brew install k6

k6 run K6_FILE.js

# Overrides
k6 run -e TARGET_URL=URL K6_FILE.js      # Environment variable
k6 run --vus 50 K6_FILE.js                              # Virtual users
k6 run --duration 60s K6_FILE.js                        # Duration

# Output
k6 run --out json=results.json K6_FILE.js               # Output to json
k6 run --out csv=results.csv K6_FILE.js                 # Output to csv
k6 run --out cloud K6_FILE.js                           # Output to cloud
k6 run --quiet K6_FILE.js                               # Suppress detailed output
```
