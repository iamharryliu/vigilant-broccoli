## Types of Cuts

| Term          | Brief Description                                                      | Image |
| ------------- | ---------------------------------------------------------------------- | ----- |
| Slice         | Cutting food into flat pieces, thin or thick depending on need.        |       |
| Chop          | Rough, quick cuts into medium or large irregular pieces.               |       |
| Dice (Large)  | Cubes about ¾ inch (2 cm).                                             |       |
| Dice (Medium) | Cubes about ½ inch (1.2 cm).                                           |       |
| Dice (Small)  | Cubes about ¼ inch (0.6 cm).                                           |       |
| Brunoise      | Very small dice made from julienne, about 1/8 inch (3 mm) cubes.       |       |
| Fine Brunoise | Extremely small dice, about 1/16 inch (1.5 mm).                        |       |
| Mince         | Very fine, tiny pieces—often used for garlic, herbs, and aromatics.    |       |
| Batonnet      | Thicker matchsticks, about ¼ inch (6 mm) thick.                        |       |
| Julienne      | Thin matchstick strips, about 1/8 inch (3 mm) thick and 2 inches long. |       |
| Chiffonade    | Thin ribbons, usually for leafy herbs or greens.                       |       |
| Bias Cut      | Diagonal slicing to increase surface area and appearance.              |       |
| Rondelle      | Circular slices typically from cylindrical vegetables like carrots.    |       |
| Paysanne      | Thin, rustic slices cut into a flat square, triangle, or round shape.  |       |
