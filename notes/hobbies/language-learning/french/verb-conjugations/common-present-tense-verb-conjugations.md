# Common Present Tense Verb Conjugations

| Subject    | -er  | -ir     | -re  |
| ---------- | ---- | ------- | ---- |
| je         | -e   | -is     | -s   |
| tu         | -es  | -is     | -s   |
| il/elle/on | -e   | -it     | -    |
| nous       | -ons | -issons | -ons |
| vous       | -ez  | -issez  | -ez  |
| ils/elles  | -ent | -issent | -ent |
