# Spanish Past Tense

| Ending | Past Tense Ending |
| :----- | :---------------- |
| -ar    | -ado              |
| -ir    | -ido              |
| -er    | -ido              |

| English           | Spanish              |
| :---------------- | :------------------- |
| I have [verb]     | he [verb]            |
| you have [verb]   | has/(usted)ha [verb] |
| he/she has [verb] | (el/ella) ha [verb]  |
| we have [verb]    | hemos [verb]         |
| they have [verb]  | han [verb]           |
