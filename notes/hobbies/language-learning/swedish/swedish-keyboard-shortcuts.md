# Swedish Keyboard Shortcuts

| Character | Shortcut                       |
| --------- | ------------------------------ |
| å         | `Option + A`                   |
| Å         | `Option + Shift + A`           |
| ä         | `Option + U`, then `A`         |
| Ä         | `Option + U`, then `Shift + A` |
| ö         | `Option + U`, then `O`         |
| Ö         | `Option + U`, then `Shift + O` |
