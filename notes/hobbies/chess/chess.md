# Chess

## Terminology

| Term                | Description                                                                    |
| :------------------ | :----------------------------------------------------------------------------- |
| Opening             | The beginning of the game, focused on development and king safety.             |
| Middlegame          | The phase where most tactics and plans occur.                                  |
| Endgame             | When few pieces remain and king activity becomes critical.                     |
| Castling            | King and rook move simultaneously for king safety.                             |
| En Passant          | Pawn capture immediately after an opponent’s two-square pawn advance.          |
| Promotion           | Pawn reaching the last rank becomes a queen, rook, bishop, or knight.          |
| Check               | King is under attack.                                                          |
| Checkmate           | King is in check and cannot escape.                                            |
| Stalemate           | Player has no legal moves but is not in check (draw).                          |
| Draw                | Game ends with no winner.                                                      |
| Fork                | One piece attacks two or more pieces at once.                                  |
| Pin                 | A piece cannot move without exposing a more valuable piece.                    |
| Skewer              | Similar to a pin, but the more valuable piece is attacked first.               |
| Discovered Attack   | An attack revealed when a piece moves out of the way.                          |
| Zwischenzug         | An intermediate move played before an expected recapture.                      |
| Sacrifice           | Giving up material for positional or tactical gain.                            |
| Development         | Bringing pieces into active positions.                                         |
| Center Control      | Dominating central squares (e4, d4, e5, d5).                                   |
| Tempo               | A single move or unit of time in development.                                  |
| Initiative          | Forcing the opponent to respond to threats.                                    |
| Space Advantage     | Controlling more squares than the opponent.                                    |
| Weak Square         | A square that cannot be defended by pawns.                                     |
| Isolated Pawn       | Pawn with no friendly pawns on adjacent files.                                 |
| Doubled Pawns       | Two pawns on the same file.                                                    |
| Passed Pawn         | Pawn with no opposing pawns blocking or capturing it.                          |
| Backward Pawn       | Pawn that cannot advance safely and is hard to defend.                         |
| File                | Vertical column (a–h).                                                         |
| Rank                | Horizontal row (1–8).                                                          |
| Diagonal            | Slanted line of squares.                                                       |
| Flight Square       | A square the king can move to in order to escape check or avoid checkmate.     |
| Intersection Square | A square where multiple lines of attack or piece paths cross.                  |
| Back Rank Mate      | Checkmate delivered on the back rank when escape squares are blocked by pawns. |
| Hookmate            | A mating pattern using a rook supported by a knight or pawn near the king.     |
| Hanging Piece       | A is a piece that is “left en prise” — it can be taken for free.               |

## Opening

- Control center
- Pawns
  - Create/challenge Pawn duos
  - Avoid blocking your pawns.
- Develop
  - Develop minor pieces early.
  - Avoid blocking your own pieces:
    - Central pawns
    - Queenside Bishop - Ideally develop queenside bishop first before playing king's pawn to avoid blocking it in.
  - Castle early.
  - Develop queen in later stages.
  - Avoid moving the same piece multiple times.

## Checkmates

- **Back Rank Mate**
  - Composition: rook
  - Opposition king stuck in back rank > rook moves to back rank and checkmates
- **Hook Mate**
  - Composition: pawn + knight + rook
  - Pawn defending knight, knight defends checking square
- **Anastasia's Mate**
  - Composition: knight + rook/queen(sacrifice) + rook
  - Knight forces king into corner square > sacrifice attack on king in end file > rook checkmates on file
- **Blind Swine Mate**
  - Composition: 2 rooks
  - Rooks on 2nd last rank > eats pieces in 2nd rank > pushes king into checkmate
- **Smothered Mate**
  - Composition: knight
  - King has no available moves > knight checkmates king

## Turn

- Check for checks > captures > threats.
- Each move should develop a piece or create/prevent a threat.
