# Skatebaord Lingo

| Term                | Description                                                                        |
| ------------------- | ---------------------------------------------------------------------------------- |
| Deck                | The wooden board you stand on. Usually made from layered maple plywood.            |
| Nose                | The front end of the deck, usually slightly longer and steeper than the tail.      |
| Tail                | The back end of the deck used for popping tricks and steering.                     |
| Kicktail            | The upward curve on the nose or tail that helps with tricks and control.           |
| Concave             | The inward curve across the width of the deck that improves foot grip and control. |
| Grip Tape           | Sandpaper-like adhesive surface applied to the top of the deck for traction.       |
| Wheelbase           | Distance between the inner mounting holes of the trucks.                           |
| Width               | Side-to-side measurement of the deck, typically in inches.                         |
| Length              | Overall measurement from nose to tail.                                             |
| Trucks              | Metal T-shaped assemblies mounted underneath the deck that hold the wheels.        |
| Hanger              | Main horizontal part of the truck that the axle passes through.                    |
| Axle                | Metal rod running through the hanger that the wheels mount onto.                   |
| Baseplate           | Flat part of the truck screwed into the deck.                                      |
| Kingpin             | Large central bolt that holds truck components together.                           |
| Bushings            | Urethane rubber rings around the kingpin.                                          |
| Pivot Cup           | Small urethane cup that stabilizes truck movement at the pivot point.              |
| Hardware            | Bolts and nuts used to attach trucks to the deck.                                  |
| Risers / Riser Pads | Pads placed between trucks and deck to increase height.                            |
| Shock Pads          | Softer riser pads designed for vibration absorption.                               |
| Wheels              | Urethane rollers attached to the trucks.                                           |
| Wheel Diameter      | Height of the wheel in millimeters.                                                |
| Durometer           | Wheel hardness rating, usually measured on the “A” scale.                          |
| Bearings            | Precision metal rings inside wheels that allow smooth spinning.                    |
| Bearing Shields     | Covers protecting bearings from dirt and debris.                                   |
| Spacer              | Small tube placed between bearings inside a wheel.                                 |
| Speed Rings         | Small washers placed between bearings and axle nuts.                               |
| Complete            | A fully assembled skateboard.                                                      |
| Setup               | A skateboard’s overall component configuration.                                    |
| Cruiser             | Skateboard designed mainly for transportation and comfort.                         |
| Longboard           | Longer skateboard designed for cruising or downhill riding.                        |
| Penny Board         | Small plastic cruiser skateboard.                                                  |
| Popsicle Deck       | Standard symmetrical skateboard deck shape.                                        |
| Old School Shape    | Wider, less symmetrical skateboard deck shape common in earlier designs.           |
