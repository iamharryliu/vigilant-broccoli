# MyWorkspace

## Commands

Test and lint.

```
ng test my-lib
ng lint my-lib
```

Build and publish library

```
cd ~/vigilant-broccoli/projects/my-workspace
ng build my-lib
cd dist/my-lib
npm publish --access public
```

Install library to project.

```
cd [project]
npm i @prettydamntired/my-lib
```
