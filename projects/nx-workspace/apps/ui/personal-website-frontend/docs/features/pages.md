# Pages

## `/` - Home

- Renders the about page content

## `/about` - About

- Markdown content from `assets/site-content/about.md`
- Badges with hover animations

## `/career` - Career

- Markdown content from `assets/site-content/career.md`

## `/contact` - Contact

- Contact form with name, email, message
- Sends via backend `/contact/send-message`

## `/links` - Link Tree

- Profile card
- External link buttons (resume, LinkedIn, GitHub, etc.)

## `/docs-md` - DocsMD

- File browser for markdown notes library
- Renders individual markdown files at `/docs-md/:filename`

## `/grind-75` - Grind 75

- LeetCode solution viewer
- Individual solutions at `/grind-75/:language/:filename`

## `/component-library` - Component Library

- Sandbox page for components
