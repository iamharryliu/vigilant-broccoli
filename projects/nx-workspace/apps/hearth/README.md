## Stack

- Next.js + React + TypeScript
- Radix Themes, lucide-react, recharts
- Supabase (auth + data)
- AWS S3 (`home-management` bucket)
- Leaflet / react-leaflet, FullCalendar
- Zod
- Internal VB libs (`common-js`, `common-node`, `react-lib`)
- Deployed to Vercel
