# iamharryliu

Personal website.

## Setup

macOS / Linux

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run.py
```

Windows

```
git clone https://github.com/iamharryliu/iamharryliu.git
cd iamharryliu
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
python run.py
```
