# Node

## General Commands

Run application.

```
npm i
npm run serve
```

Run application tests.

```
npm run test
```

## NPM Commands

Build and publish.

```
npm run build && npm publish
```

Link and Unlink NPM Packages

```
cd [package]
npm link
cd [project]
npm unlink --no-save [package-name]
```
